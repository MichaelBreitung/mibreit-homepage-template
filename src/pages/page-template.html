I use this page template to explain the different sections and how to structure your content using HTML markup. To better read this document, copy this page into your "src/pages" folder and build your homepage. Then you can use Laragon or XAMPP to locally host your homepage and access the rendered template in your browser by opening "http://localhost/page-template.html".

Content Block

The content block is the location where you put the content of the page. It will mostly be text. But you can also include images, slideshows, galleries, lists, tables - whatever the HTML markup allows. If you want to use a slideshow or gallery, you must change line 7 of this template and use the "slideshow-content-layout.njk" or "gallery-layout.njk" instead of the "content-layout.njk". In that case, you must also change the page extension from "html" to "php". For all other content, prefer to use "html".

Content Structure

You might already have noticed the tags I'm using in this template: "h1", "h2", "p". Those tags structure your content. Beyond that, they are very important for SEO - Search Engine Optimization.

Notice that whenever I use an "opening" tag on this page, I must also close it. For the "h1" headline, the opening and closing pair are <h1>headline</h1>. It is important to not forget the closing tags. Otherwise, your page will not render properly.

I recommend to always validate the code of your page using the W3C checker immediately after you have published it. If the validator finds errors, correct them.

h1

The "h1" tag is for the main headline. You should only have one "h1" tag in your page. Provide a good title for the page. You can use the same or a similar title to what you have used on line 1 of this template. Make sure to include your main keywords for which you want to rank on Google. A good guide on page titles can be found on the Fuel Your Photos homepage.

h2

As the name suggests, the "h2" tag is used for secondary headlines - section headlines. You should use "h2" headlines to provide structure to longer content. It will not only help your readers, but also Google to identify what your page is about. It helps with better ranking. The "h2" headlines are a good place to include further keywords for which you want to rank.

h3

If you find you need to structure your sections even more, you can use "h3" tags to do so. Those are helpful if you write content of more than 1000 words.

p

The "p" tag is the one you will use the most on your pages - "p" stands for paragraph. Every paragraph you read on this template page is enclosed in "p" tags. In general, all the text on your page should be enclosed in either a set of headline tags or paragraph tags.

a

I provide links to other homepages on this template page. I do so, by enclosing the link texts in "a" tags. "a" tags allow you to provide attributes in he opening tag. The most important attribute is the "href" attribute. Explore the code of this template to see how it is used. For external links, it will contain the complete URL of the target page. For internal links to other pages on your homepage, use relative links starting with "/". For example "/contact.html".

There is a second, very important attribute you should use for external links, which is "target". Always set it to "_blank" for external sites so they open in a new tab in your browser and your homepage stays active. You don't want a visitor to directly leave your page when clicking on an external link.

For internal pages, it's up to you to decide if you want them to open in a separate tab or not. Mostly, you don't want this and leave out the "target" attribute for internal links.

You can read more about the "a" tag here. Notice how the previous link says just "here". This is bad practice. You should try to have a more descriptive link text.

img

The "img" tag is used wherever you want to display an image on the page. It is different to the use of images within a gallery or slideshow. The "img" tag will simply display the image right where you place it within the text. You should always use it outside of a paragraph, so the styling works properly.
